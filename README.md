# Downloads Organization Script

This repository contains scripts for organizing downloads and managing file organization tasks.

## Key Points

1. **Central Scripts Location**

   - All scripts are stored in `~/Scripts/`
   - Provides a permanent, organized location for script management

2. **Easy Access via Alias**

   - Use the `organize` alias to run the script
   - Points to `~/Scripts/organize_downloads.sh`

3. **Self-Management**

   - Automatically moves all `.sh` files to the scripts folder
   - Includes self-relocation capability

4. **Global Accessibility**
   - Run the `organize` command from any directory
   - No need to specify full path

## Usage

Simply type `organize` in your terminal to run the script.

## Installation

Make sure the script is placed in `~/Scripts/` and the alias is properly configured in your shell configuration file.
